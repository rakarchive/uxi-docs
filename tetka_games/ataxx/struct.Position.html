pub struct Position {
    pub bitboards: [BitBoard; 3],
    pub checksum: Hash,
    pub side_to_move: Color,
    pub ply_count: u16,
    pub half_move_clock: u8,
}
Expand description

Position represents the snapshot of an Ataxx Board, the state of the an ataxx game at a single point in time. It also provides all of the methods necessary to manipulate such a snapshot.

Fields§

§bitboards: [BitBoard; 3]

bitboards stores BitBoards for the piece configuration of each piece.

§checksum: Hash

checksum stores the semi-unique Hash of the current Position.

§side_to_move: Color

side_to_move stores the piece whose turn to move it currently is.

§ply_count: u16§half_move_clock: u8

half-move clock stores the number of half-moves since the last irreversible Move. It is used to adjudicate games using the 50-move/100-ply rule.
