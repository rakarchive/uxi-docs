unsafe fn unsafe_from<T: Copy + Into<usize>>(number: T) -> Self

unsafe_from unsafely converts the given number into Self.

§Safety

unsafe_from assumes that the target type can represent the provided number, i.e. the number has a valid representation in the target type. The function comes with a debug check for the same, and failure to uphold this invariant will result in undefined behavior.

Object Safety§

This trait is not object safe.
