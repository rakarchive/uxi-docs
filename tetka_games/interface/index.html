    BitBoardType is a generalized interface implemented by BitBoards of arbitrary size. This allows programs to handle BitBoards of any size with generic functions using this common interface.

  • MoveType

  • PositionType
    Position is a generalized interface for board representations of a wide range of games. It can be used to create game-agnostic software. Tetka provides some of the popular board representations out of the box, but custom ones can also be implemented by the library user.
  • RepresentableType
    RepresentableType is a basic trait which is implemented by enums with both a binary and string representation and backed by an integer.
  • SquareType
    SquareType is the trait implemented by the type for Squares in a game’s board representation. It provides a generic interface along with useful predefined methods that a Square type would require.
